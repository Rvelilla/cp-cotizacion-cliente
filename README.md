# Generador de Cotizaciones - Carrocerías Panamericana

Este proyecto automatiza la transformación de cotizaciones en formato PDF a documentos de Word editables, permitiendo personalizar el asesor y los detalles técnicos.

## Instalación

1. Asegúrate de tener Python instalado.
2. Instala las librerías necesarias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

1. Ejecuta la aplicación de Streamlit:
   ```bash
   streamlit run app.py
   ```
2. Sube el PDF de la cotización base.
3. Selecciona el asesor y genera el documento Word.

_Nota: Asegúrate de que la carpeta `assets/` y `plantillas/` contengan los archivos necesarios antes de iniciar._
